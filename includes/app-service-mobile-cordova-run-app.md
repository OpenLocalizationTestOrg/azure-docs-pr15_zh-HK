
1. 请访问[Azure 的门户]。 单击**浏览所有** > **移动应用程序**> 您刚刚创建的后端。 在移动应用程序设置中，单击**快速启动** > **Cordova**。 在**配置客户端应用程序**，选择**创建新的应用程序**，然后单击**下载**。 这会将下载完整的 Cordova 项目预配置为连接到您的后端应用程序。

2. 解压缩下载的 ZIP 文件的目录在硬盘上，定位到解决方案文件 (.sln)，使用 Visual Studio 将其打开。

5. 在 Visual Studio 中，从开始箭头，旁边的下拉列表中选择解决方案平台 （Android、 iOS 或窗口），然后通过单击绿色箭头上的下拉列表中选择特定部署设备或仿真程序。 注意，您可以使用默认的 Android 平台和波纹仿真程序。 更多高级的教程将要求您选择一个受支持的设备或仿真程序。 

6. 请按 F5 或单击绿色箭头来构建和运行 Cordova 应用程序。 如果您看到在网络仿真程序请求访问一个安全对话框，接受它。   

7. 后启动该应用程序在设备或仿真程序上，**输入新的文本**，例如_完成本教程_中键入有意义的文本，然后单击**添加**按钮。  
这将 POST 请求发送到 Azure 的后端早部署。 从请求的后端插入数据到 TodoItem 表在 SQL 数据库中，并返回到移动应用程序的新存储项目的信息。 移动应用程序在列表中显示此数据。

    ![](./media/app-service-mobile-cordova-quickstart/quickstart-startup.png)
    
8. 要支持的每个设备平台重复以前的三个步骤。

[Azure 门户]: https://portal.azure.com/
