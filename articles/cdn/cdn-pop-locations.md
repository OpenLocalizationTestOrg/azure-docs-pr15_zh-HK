>[AZURE.IMPORTANT] \***Akamai 从 azure CDN**POP 位置不单独披露。  
>
>这两个提供商有不同方法可以建立其 CDN 基础结构。  我们建议不要使用 POP 的位置来决定哪种 Azure CDN 产品使用，并考虑功能和最终用户性能。  测试两个提供程序选择适当的 Azure CDN 产品为您的用户具有的性能。
